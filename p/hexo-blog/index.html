Hexo | 搭建博客并上传至Github

Created2022-09-09|Updated2022-12-10|Word count:898|Reading time:3min|Post View:|Comments:

准备

1、一台电脑

2、[选用]域名

3 、Github账户点此注册

安装环境

安装Git

安装Node.js

安装检验

使用Hexo搭建博客(本地启动)

安装Hexo

生成初始博客文件

在任意文件夹进入cmd(在文件夹路径输入cmd)

生成blog文件夹:

本地启动

进入blog文件夹:

启动:

出现 ‘hexo is running at http://localhost:4000，press ctrl+c to stop’

即表示成功

可以通过 http://localhost:4000/ 来访问

若出现乱码，尝试将_config.yml的文件编码格式改为UTF-8

配置博客内容

更改基本信息

用编辑器打开_config.yml，修改相应的内容

  • title(博客标题)

  • subtitle(子标题)

  • description(博客描述)

  • keywords(关键字，使用半角逗号，分隔多个关键字)

  • author(作者)

  • language(语言)

  • timezone(网站时区，默认使用本机的时区)

修改完后保存退出

更改主题

先停止运行博客

在 Hexo官网 找想要的主题

找到后点击预览图进入预览界面或点击标题进入项目界面

在项目界面内根据主题的步骤

安装完成后进入blog根目录下的_config.yml，找到theme，

修改成主题的名称

启动:

通过 http://localhost:4000/ 就可以访问博客

由于各主体配置文件内容不同，所以请前往主题的官网，这里不多阐述

进入Github中你的储存库的设置

点击 pages ，找到 Build and deployment

将 branch 小标题下的分支改为 master ，也就是部署博客的分支

点击 save

通过 github用户名.github.io 即可访问

域名

如果觉得Github提供的域名不美观，可以自定义域名

注册

首先按照域名注册完成域名注册

解析域名

按照域名解析完成解析

DNS记录

其中,DNS记录填写以下值

  • 类型：CNAME

  • 名称(即子域名)：任意子域名(如 www、blog 等)

  • 目标：github用户名.github.io

Github配置域名

返回Github储存库

点击 settings –> page ，找到 custom domain

输入域名(需要加上子域名)

等待DNS检查成功后即可访问
